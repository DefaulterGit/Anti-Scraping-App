Anti-ScrApp - сервис, предназначенный для противодействия WEB-скрапингу.
Функциональные требования:
1) Anti-ScrApp - прокси-сервис
2) Перехват запросов от пользователей к защищаемому сайту
3) Анализ запросов для обнаружения маркеров скраперов
4) Использование различных методов (CAPTCHA, User Behaviour Analytics, генерация математических примеров) для выявления скраперов
5) Админ-панель:
	- настройка различных параметров сервиса
	- просмотр и редактирование списка заблокированных клиентов, а также белого списка скраперов
	- статистика: система должна предоставлять графики и отчеты по количеству заблокированных запросов, источников трафика и загруженности сервера
6) Двухфакторная аутентификация с подтверждением по Google Authenticator
7) Используется реляционная БД
8) Присвоение пользователям "рейтинга подозрительности" (Risk Score)
9) Временная блокировка подозрительного клиента (с таймером автоматического снятия)
10) Блокировка осуществляется не только по IP, но и по цифровому отпечатку браузера (TLS fingerprint, порядок HTTP-заголовков)
11) Cookie валидация: После успешного прохождения проверки (например, решения CAPTCHA), система должна устанавливать в браузер пользователя специальный зашифрованный cookie-файл, разрешающий доступ без проверок в течение определенной сессии
